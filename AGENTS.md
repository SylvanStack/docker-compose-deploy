# OpenSpec 指南

这些指南适用于在此项目中工作的 AI 助手。

当请求满足以下条件时，始终打开 `@/openspec/AGENTS.md`：
- 提到规划或提案（如 proposal、spec、change、plan 等词）
- 引入新功能、破坏性变更、架构转变或重大性能/安全工作
- 听起来模糊不清，需要在编码前查看权威规范

使用 `@/openspec/AGENTS.md` 来学习：
- 如何创建和应用变更提案
- 规范格式和约定
- 项目结构和指南

保留此管理块，以便 'openspec update' 可以刷新指南。
